zMD Help

Getting Started

zMD is a lightweight macOS markdown viewer for clean reading and professional document export.

Opening Files

  • Press ⌘O to open files
  • Drag and drop .md files onto the app
  • Double-click markdown files in Finder
  • Access recent files from File → Open Recent

Working with Tabs

  • ⌘O - Open new files in tabs
  • ⌘W - Close current tab
  • ⌃Tab - Next tab
  • ⌃⇧Tab - Previous tab

Customizing Appearance

Press ⌘, to open Settings.

Theme Selection

  • System - Follows macOS appearance
  • Light - Always light mode
  • Dark - Always dark mode

Font Styles

  • System - San Francisco font
  • Serif - Charter font
  • Monospace - Monospaced font

Exporting Documents

Export via File → Export:

  • PDF - Formatted with pagination
  • HTML - With or without styles
  • Word (.docx) - Microsoft Word format

File Management

  • Open File Location - Reveal in Finder
  • Duplicate (⌘⇧S) - Copy file
  • Rename - Rename document
  • Move To - Move to another folder
